# Atlas AI Chatbot Prototype

Atlas is a modern AI chatbot prototype built with React, TypeScript, and Vite.

## Features
- **Modern UI:** Clean, responsive design with a professional aesthetic.
- **Atlas Persona:** A friendly and capable AI assistant named Atlas.
- **Real-time Interaction:** Simulated typing indicators and dynamic responses.

## Getting Started
To get started with the project, follow these steps:

1.  **Install dependencies:**
    ```bash
    npm install
    ```
2.  **Start the development server:**
    ```bash
    npm run dev
    ```

## Project Structure
- `src/App.tsx`: Main application component and chatbot logic.
- `src/App.css`: Styles for the chat interface.
- `public/ai-logo.png`: The logo for Atlas.
